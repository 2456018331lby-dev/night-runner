# Night Runner Design Brief

## 核心主题

霓虹都市夜晚追逐。主角是被追捕的跑者，在屋顶、施工栈桥和列车之间高速穿梭，通过冲刺和击飞解决拦截者。

## 核心情绪

- 快
- 危险
- 帅气
- 短局可重复

## 核心循环

1. 前进并抢走屋顶上的情报核心
2. 用冲刺和跳跃穿过火力线
3. 近身把敌人击落平台，维持连击
4. 核心拿齐后决定继续刷分还是立刻撤离

## 为什么适合安卓 + Steam

- 安卓：操作动作少、回合短、性能压力低
- PC：可扩关卡、Boss、评分、Steam 成就、手柄和挑战模式

## 版本路线

### V0 当前原型

- 单关卡循环
- 角色基础机动
- 情报核心收集、撤离点和结算评级
- 近战和击落得分
- 两种敌人原型（追击者 / 远程压制者）

### V1 安卓测试版

- 3 个短关
- 2 到 3 类敌人
- 更好的触屏布局和震动反馈
- 音效、打击停顿、命中闪白

### V2 PC / Steam 预备版

- 章节结构
- 关卡评分
- Boss
- 存档与解锁
- 手柄、窗口设置、Steamworks 接口层
